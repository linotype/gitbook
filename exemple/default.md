# Default

### Install

```text
composer require linotype/starter
```

### The **default** **starter** contain: 

* **Title** block
* **Text** field
* **Hello** helper
* **Header** module
* **Footer** module
* **Page** template
* **Default** theme

{% hint style="info" %}
**linotype/starter** based on stimulus frontend library  
{% endhint %}
